## 🎮 交互演示

### 场景 1：首次访问

```
用户打开网站
    ↓
看到 3D 画廊主页
    ↓
画廊空空如也
    ↓
显示"立即上传"按钮

点击按钮跳转到上传页面
```

### 场景 2：上传照片

```
在上传页面
    ↓
拖拽照片到上传区域
    ↓
看到图片预览
    ↓
添加描述"美丽的风景"
    ↓
点击"上传到画廊"
    ↓
显示"上传成功"提示
    ↓
2秒后自动跳转到画廊
    ↓
看到新照片在 3D 画廊中展示
```

### 场景 3：浏览照片（3D 视图）

```
在 3D 画廊页面
    ↓
移动鼠标
    ↓
照片跟随鼠标倾斜（很酷！）
    ↓
鼠标悬停在某张照片上
    ↓

打开照片详情模态框
```


## 🎨 视觉效果说明

### 动画效果

1. **页面切换**
   ```
   当前页面 ──► 淡出(0.5s) ──► 新页面淡入(0.5s)
   ```

2. **3D 卡片浮动**
   ```
   位置A ──► 上浮20px(1.5s) ──► 下沉(1.5s) ──► 位置A
                ↺ 无限循环
   ```

3. **悬停效果**
   ```
   普通状态 ──► 倾斜10° + 放大1.05倍(0.6s) ──► 悬停状态
   ```

4. **模态框弹出**
   ```
   隐藏 ──► 缩放0.8→1.0 + 淡入(0.4s) ──► 显示
   ```

### 颜色方案

```css
/* 主色调 */
主要: #667eea (亮紫) → #764ba2 (深紫)
成功: #2ecc71 (绿色)
错误: #e74c3c (红色)
警告: #f39c12 (橙色)

/* 中性色 */
深色: #2c3e50
灰色: #95a5a6  
浅色: #ecf0f1
白色: #ffffff
```

### 间距系统

```

大间距: 32px
超大: 64px
```

## 🎯 核心代码说明

### HTML 结构

```html
<nav>导航栏</nav>

<section id="galleryPage" class="active">
    3D 画廊内容
</section>

<section id="uploadPage">
    上传页面内容
</section>

<div id="photoModal">
    照片详情模态框
</div>
```

### CSS 关键技术

```css
/* 3D 变换 */
transform: rotateY(10deg) rotateX(-5deg) scale(1.05);
transform-style: preserve-3d;
perspective: 2000px;

/* 毛玻璃效果 */
backdrop-filter: blur(10px);
background: rgba(255, 255, 255, 0.95);

/* 渐变背景 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 动画 */
@keyframes float {
    0%, 100% { transform: translateY(0); }
    50% { transform: translateY(-20px); }
}
```
