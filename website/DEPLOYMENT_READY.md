## 🔐 安全提示

在生产环境部署前：

1. **修改默认密码**
   ```json
   {
     "username": "cecilia",
     "password": "修改为强密码"
   }
   ```

2. **修改SECRET_KEY**
   在 `app.py` 中或通过环境变量设置：
   ```python
   app.config['SECRET_KEY'] = '生成一个随机密钥'
   ```

3. **使用HTTPS**
   - 配置nginx反向代理
   - 申请SSL证书

4. **限制访问**
   - 配置防火墙
   - 限制IP访问范围

5. **定期备份**
   - 备份 `uploads/` 目录
   - 备份 `metadata.json`
   - 备份 `accounts.json`
