# 📺 全屏画廊使用指南

## 🎯 设计理念

全屏沉浸式画廊采用极简设计，让您专注于照片本身：

- **一次一张** - 每次只展示一张照片
- **自适应** - 照片自动适配屏幕大小
- **无干扰** - 黑色背景，极简界面
- **多种操作** - 支持多种浏览方式

## 🖼️ 界面布局

```
╔════════════════════════════════════════╗
║  导航栏（顶部）                        ║
╠════════════════════════════════════════╣
║                                        ║
║   [←]                           [→]   ║
║           [  照片自适应展示  ]         ║
║                                        ║
║                                        ║
╠════════════════════════════════════════╣
║  3 / 10    [▶自动播放] [🗑️删除]       ║
╚════════════════════════════════════════╝
```

**界面元素**：
- 顶部：导航栏（上传入口、照片计数）
- 中央：照片全屏展示区域
- 左侧：上一张箭头
- 右侧：下一张箭头
- 底部：照片计数 + 控制按钮

## 🎮 操作方式

### 1. 鼠标操作

#### 切换照片
- 点击**左侧箭头**（‹）：上一张
- 点击**右侧箭头**（›）：下一张
- **鼠标滚轮向下**：下一张
- **鼠标滚轮向上**：上一张

#### 控制功能
- 点击 **▶** 按钮：开始自动播放
- 点击 **⏸** 按钮：暂停自动播放
- 点击 **🗑️** 按钮：删除当前照片

### 2. 键盘操作

| 按键 | 功能 |
|------|------|
| **←** 左方向键 | 上一张照片 |
| **→** 右方向键 | 下一张照片 |
| **空格** | 开始/暂停自动播放 |

### 3. 触摸操作（移动端）

- **向左滑动** → 下一张照片
- **向右滑动** → 上一张照片

## 🎬 使用场景

### 场景 1: 快速浏览

```bash
1. 打开画廊
   ↓
2. 看到第一张照片
   ↓
3. 按 → 键快速翻页
   ↓

```

### 场景 2: 自动播放

```bash
1. 点击底部 ▶ 按钮
   ↓
2. 照片每 3 秒自动切换
   ↓
3. 按空格键或点击 ⏸ 暂停
```

### 场景 3: 删除照片

```bash
1. 浏览到要删除的照片
   ↓
2. 点击底部 🗑️ 按钮
   ↓
3. 确认删除
   ↓
4. 自动显示下一张照片
```

### 场景 4: 移动端浏览

```bash
1. 在手机上打开
   ↓
2. 照片自适应屏幕
   ↓
3. 左右滑动翻页
   ↓
4. 流畅的触摸体验
```

## ⭐ 特色功能

### 1. 智能自适应

照片会自动调整大小以适应屏幕：
- **横向照片**：宽度适配
- **竖向照片**：高度适配
- **保持比例**：不变形
- **居中显示**：完美展示

### 2. 循环浏览

- 最后一张 → 按右箭头 → 回到第一张
- 第一张 → 按左箭头 → 跳到最后一张

### 3. 流畅动画

照片切换时的淡入淡出效果：
- 当前照片淡出（0.15秒）
- 新照片淡入（0.3秒）
- 缩放动画（从0.9到1.0）

### 4. 多设备支持

| 设备 | 主要操作 | 辅助操作 |
|------|----------|----------|
| 桌面 | 键盘方向键 | 点击箭头、滚轮 |
| 平板 | 触摸滑动 | 点击箭头 |
| 手机 | 触摸滑动 | 点击箭头 |

## 📊 性能说明

### 加载优化
- 只加载当前显示的照片
- 预加载前后各一张（计划中）
- 图片懒加载

### 内存管理
- 不保存历史图片在内存
- 及时释放未使用的图片资源

### 流畅度
- GPU 加速的动画
- 优化的事件处理
- 防抖和节流机制

## 🔧 自定义设置

### 修改自动播放间隔

编辑 `static/script.js`，找到：

```javascript
autoPlayInterval = setInterval(function() {
    showNextPhoto();
}, 3000); // 改为你想要的毫秒数
```

### 修改动画速度

编辑 `static/style.css`，找到：

```css
.fullscreen-image {
    transition: opacity 0.3s ease-in-out;
}
```

### 修改箭头大小

编辑 `static/style.css`，找到：

```css
.nav-arrow {
    width: 60px;  /* 改为你想要的大小 */
    height: 60px;
}
```

## ⚠️ 注意事项

1. **照片数量**
   - 建议 < 500 张以获得最佳性能
   - 大量照片时加载可能稍慢

2. **照片尺寸**
   - 建议 < 5MB 每张
   - 超大照片可能加载较慢

3. **浏览器兼容**
   - 现代浏览器完美支持
   - IE11 可能有部分功能限制

4. **网络环境**
   - 本地网络：流畅
   - 远程访问：取决于带宽

## 📞 常见问题

### Q: 如何同时看到多张照片？

A: 当前版本专注于单张展示。如需预览多张，可以使用网格视图模式（计划中）。

### Q: 可以显示照片信息吗？

A: 当前版本为极简设计，不显示文件名和描述。专注于照片本身。

### Q: 自动播放会一直循环吗？

A: 是的，播放到最后一张会自动跳回第一张继续播放。

### Q: 删除照片需要确认吗？

A: 是的，点击删除按钮会弹出确认对话框，防止误删。
