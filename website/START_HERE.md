### 常见问题
- **无法启动？** 检查Python和Flask是否安装
- **登录失败？** 确认用户名密码是否正确
- **无法上传？** 确认是否已登录，刷新浏览器缓存

### 快速修复
```bash
# 重装依赖
pip install -r requirements.txt

# 清除浏览器缓存
Ctrl + Shift + R

# 查看日志
tail -f logs/app.log
```
