# 📝 更新日志

## [2.0.0] - 2025-12-26 - 沉浸式 3D 画廊版本

### ✨ 新增功能

#### 🎨 沉浸式 3D 画廊
- **3D 视图模式**：照片以 3D 卡片形式浮动展示
  - 自然的浮动动画效果
  - 鼠标跟随倾斜交互
  - 悬停放大效果
  - GPU 加速的 3D 变换

- **网格视图模式**：经典网格布局
  - 快速浏览大量照片
  - 响应式列数调整
  - 悬停上浮效果

- **轮播视图模式**：3D 圆形轮播
  - 照片环形排列
  - 自动旋转展示
  - 立体透视效果

- **视图切换控制**：右下角浮动按钮组
  - 一键切换三种视图
  - 当前视图高亮显示
  - 毛玻璃效果背景

#### 🧭 智能导航系统
- **固定顶部导航栏**
  - 毛玻璃半透明效果
  - 品牌 Logo 和名称
  - 页面切换按钮
  - 实时照片计数
  - 响应式移动端菜单

- **页面分离**
  - 独立的画廊浏览页面（默认）
  - 独立的照片上传页面
  - 平滑的页面切换动画

#### 📤 优化的上传体验
- **拖拽上传**
  - 支持拖拽文件到上传区域
  - 拖拽悬停状态提示
  - 友好的视觉反馈

- **图片预览**
  - 上传前实时预览照片
  - 大图预览展示
  - 一键移除重选

- **智能跳转**
  - 上传成功后自动跳转到画廊
  - 2秒延迟，显示成功提示

- **优化的上传界面**
  - 大图标和清晰提示
  - 文件格式说明
  - 现代化卡片设计

#### 🖼️ 增强的照片详情
- **全新模态框设计**
  - 左侧大图展示
  - 右侧信息面板
  - 磨砂玻璃背景

- **前后翻页功能**
  - 左右导航按钮
  - 键盘方向键支持
  - 循环浏览照片

- **优化的关闭方式**
  - 点击背景关闭
  - 点击 X 按钮关闭
  - ESC 键快速关闭

## 版本对比

### 1.0 → 2.0 主要改进

| 功能 | 1.0 | 2.0 |
|------|-----|-----|
| 浏览模式 | 单一网格 | 3D/网格/轮播 三种 |
| 页面结构 | 单页滚动 | 分页导航 |
| 上传方式 | 点击选择 | 拖拽 + 选择 + 预览 |
| 照片详情 | 基础模态框 | 增强模态框 + 翻页 |
| 导航系统 | 无 | 固定导航栏 |
| 3D 效果 | 无 | 完整 3D 支持 |
| 键盘操作 | ESC 关闭 | 完整键盘导航 |
| 动画效果 | 基础 | 丰富动画系统 |
| 视觉设计 | 简单 | 现代化设计 |
